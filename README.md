# Lab Website

[Website](https://ntut-sslab.github.io/)

## Main File Structure

- `docs/`: 網站根目錄
	- `advisor/`: 教授相關資訊
		- `certification.yaml`: 教授取得的證照
		- `paper.yaml`: 教授發表或參與的期刊論文
	- `avatar/`: 大頭照
	- `i18n/`: 翻譯相關檔案
	- `image`: 照片
	- `member/`: 成員資訊
		- `graduated.json`: 已畢業成員
		- `students.json`: 未畢業成員
	- `research/`: 研究與產學計畫
		- `abstract/`: 研究摘要
		- `research.json`: 研究與產學計畫資訊
	- `thesis.json`: 學生畢業論文
